🎨 CSS Frameworks Recomendados

Bulma CSS:

  • ✅ Flexbox-based - Perfecto para layouts modernos
  • ✅ Modular - Solo importar lo necesario
  • ✅ Mobile-first - Responsive por defecto
  • ✅ Componentes listos - Navbar, cards, modals
  • ✅ Customizable - Variables SASS

Alternativas: Tailwind CSS (utility-first), Bootstrap 5 (familiar)
